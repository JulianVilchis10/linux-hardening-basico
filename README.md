# Linux Hardening Básico

Repositorio de laboratorio personal enfocado en prácticas básicas de hardening
y seguridad en sistemas Linux.

## Objetivo
Aplicar configuraciones de seguridad iniciales para reducir la superficie de ataque
en un sistema Linux.

## Contenido
- Gestión de usuarios y permisos
- Configuración segura de SSH
- Firewall con UFW
- Revisión de logs y auditoría básica

## Entorno
- Distribución: Kali
- Usuario no root
- Entorno de laboratorio local
